  <files>src/DX.Blame.Hg.Discovery.pas</files>
  <action>
Create src/DX.Blame.Hg.Discovery.pas mirroring the exact structure of DX.Blame.Git.Discovery.pas.

**Interface section exports:**
- `function FindHgExecutable: string;` — finds hg.exe on the system
- `function FindHgRepoRoot(const APath: string): string;` — finds .hg repo root
- `procedure ClearHgDiscoveryCache;` — resets cached values (called on project switch)

**FindHgExecutable implementation:**
Search order (exit on first hit):
1. System PATH — `System.SysUtils.GetEnvironmentVariable('PATH')`, split by ';', check each dir for `hg.exe`
2. TortoiseHg default: `C:\Program Files\TortoiseHg\hg.exe`
3. TortoiseHg x86: `C:\Program Files (x86)\TortoiseHg\hg.exe`

Cache result in unit-level vars `GCachedHgPath: string` and `GHgPathSearched: Boolean`, same pattern as Git.Discovery.

Note: Research suggested also checking registry HKLM\SOFTWARE\TortoiseHg for InstallDir, but this was marked LOW confidence and TortoiseHg may not write that key. Omit registry for now — the PATH + default dirs cover the standard cases.

**FindHgRepoRoot implementation:**
1. If cached for same source path, return cached result
2. Determine starting directory (use TPath.GetDirectoryName if APath is a file)
3. Walk parent directories upward, checking for `.hg` subdirectory with `TDirectory.Exists`
4. When `.hg` found: call `FindHgExecutable`. If hg.exe found, verify with `hg root` via `TVCSProcess.Create(LHgPath, LDir)` then `Execute('root', LOutput)`. Exit code 0 = valid, use LDir as repo root (NOT hg root output, to avoid UNC vs drive letter mismatch per Pitfall 3 in research).
5. If hg.exe NOT found when .hg exists, return empty string (unlike Git which has a fallback — Mercurial without hg.exe is unusable per research)
6. Cache result in unit-level vars `GCachedHgRepoRoot` and `GCachedHgRepoRootSource`

**ClearHgDiscoveryCache:** Reset all four cache vars.

**Uses clause (implementation):** System.SysUtils, System.IOUtils, DX.Blame.VCS.Process

**File format:** UTF-8 with BOM, CRLF line endings. Include standard unit header with MIT license, Copyright 2026 Olaf Monien.
