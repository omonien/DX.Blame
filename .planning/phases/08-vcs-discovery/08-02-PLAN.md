**Part B: Create DX.Blame.VCS.Discovery.pas**

**Interface section:**
```pascal
type
  TVCSType = (vtNone, vtGit, vtMercurial);

  TVCSDiscovery = class
  public
    class function DetectProvider(const AProjectPath: string;
      out ARepoRoot: string): IVCSProvider;
  end;
```

**Implementation uses:** System.SysUtils, System.IOUtils, Vcl.Dialogs, Vcl.Controls, DX.Blame.VCS.Provider, DX.Blame.Git.Provider, DX.Blame.Hg.Provider, DX.Blame.Settings

**Private class methods (in implementation section, not exposed in interface):**

1. `class function ScanForVCS(const APath: string; out AHasGit, AHasHg: Boolean): Boolean;`
   - Walk parent directories upward from APath (same parent-walk pattern as FindGitRepoRoot)
   - At each level, check for `.git` AND `.hg` subdirectories (check both, do not stop at first)
   - Set AHasGit/AHasHg accordingly
   - Return True if either found
   - IMPORTANT: Both .git and .hg could be at different levels. Scan until both checked or root reached. Specifically: walk upward, at each dir check for .git and .hg. If .git found, set AHasGit. If .hg found, set AHasHg. Stop walking once BOTH are found or parent dirs exhausted.

2. `class function ResolveChoice(const AProjectPath: string; AHasGit, AHasHg: Boolean): TVCSType;`
   - If only one VCS: return that type directly
   - If both: check `BlameSettings.GetVCSChoice(AProjectPath)`
     - If persisted choice is 'Git' -> return vtGit
     - If persisted choice is 'Mercurial' -> return vtMercurial
     - If empty: call PromptForVCS, persist result, return it
   - If neither: return vtNone

3. `class function PromptForVCS(const AProjectPath: string): TVCSType;`
   - Use `Vcl.Dialogs.TaskDialog` (TTaskDialog component):
     - Title: 'DX.Blame - Multiple VCS Detected'
     - Text: 'Both Git and Mercurial repositories were found for this project. Which VCS should DX.Blame use for blame annotations?'
     - CommonButtons: [] (no common buttons)
     - Add two TTaskDialogButtonItem entries: 'Use Git' (ModalResult=100) and 'Use Mercurial' (ModalResult=101)
     - MainIcon: tdiInformation
     - Execute the dialog
   - If user clicks 'Use Git' (ModalResult=100): persist 'Git' via `BlameSettings.SetVCSChoice`, return vtGit
   - If user clicks 'Use Mercurial' (ModalResult=101): persist 'Mercurial' via `BlameSettings.SetVCSChoice`, return vtMercurial
   - If user closes dialog without choosing: default to vtGit (safe default, preserves existing behavior)

**DetectProvider implementation:**
```
1. ScanForVCS(AProjectPath, LHasGit, LHasHg)
2. LChoice := ResolveChoice(AProjectPath, LHasGit, LHasHg)
3. Case LChoice of:
   vtGit: Result := TGitProvider.Create; ARepoRoot := Result.FindRepoRoot(AProjectPath)
   vtMercurial: Result := THgProvider.Create; ARepoRoot := Result.FindRepoRoot(AProjectPath)
   vtNone: Result := nil; Exit
4. Validate: if Result.FindExecutable = '' or ARepoRoot = '' then Result := nil
```
