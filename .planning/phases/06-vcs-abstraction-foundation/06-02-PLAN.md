**Part A: Create DX.Blame.Git.Provider.pas**

Create a new unit with UTF-8 BOM and CRLF. Standard project header (XML doc, MIT, 2026 Olaf Monien).

TGitProvider = class(TInterfacedObject, IVCSProvider) — a thin delegation wrapper. It does NOT contain new logic; it delegates to existing Git units.

Interface uses clause: DX.Blame.VCS.Types, DX.Blame.VCS.Provider, Winapi.Windows.
Implementation uses clause: DX.Blame.Git.Types, DX.Blame.Git.Process, DX.Blame.Git.Discovery, DX.Blame.Git.Blame, System.SysUtils.

Method implementations:
- ExecuteBlame: Create a TGitProcess(FindGitExecutable, ARepoRoot), call ExecuteAsync with blame args ('blame --line-porcelain -- "' + ARelativePath + '"' where ARelativePath is computed from ARepoRoot and AFilePath), return exit code. Use the same argument format as DX.Blame.Engine currently uses.
- ParseBlameOutput: Delegate to DX.Blame.Git.Blame.ParseBlameOutput(AOutput, LLines) and return LLines. Note: read the current ParseBlameOutput signature from Git.Blame.pas to get the exact calling convention (it takes var parameter, not function return).
- GetCommitMessage: Create TGitProcess, execute 'log -1 --format=%B {hash}', return output. Read DX.Blame.CommitDetail.pas to see the exact git command currently used.
- GetFileDiff: Create TGitProcess, execute the same git show/diff command currently in CommitDetail for file diff. Read CommitDetail.pas for exact args.
- GetFullDiff: Same approach for full diff.
- GetFileAtRevision: Create TGitProcess, execute 'show {hash}:{relpath}'. Read DX.Blame.Navigation.pas for exact args.
- FindExecutable: Delegate to DX.Blame.Git.Discovery.FindGitExecutable.
- FindRepoRoot: Delegate to DX.Blame.Git.Discovery.FindGitRepoRoot.
- ClearDiscoveryCache: Delegate to DX.Blame.Git.Discovery.ClearDiscoveryCache.
- GetDisplayName: Return 'Git'.
- GetUncommittedHash: Return cUncommittedHash from Git.Types.
- GetUncommittedAuthor: Return cNotCommittedAuthor from Git.Types.

IMPORTANT: Read the actual source of Engine.pas, CommitDetail.pas, Navigation.pas, and Git.Blame.pas BEFORE implementing TGitProvider to get the exact git command arguments and calling conventions. Do not guess — match the existing calls exactly.
