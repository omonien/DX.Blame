**GREEN -- Diff dialog form:**

Create `src/DX.Blame.Diff.Form.pas` and `src/DX.Blame.Diff.Form.dfm`:
- `TFormDXBlameDiff = class(TForm)` with DFM.
- DFM layout (design-time):
  - Form: Caption = 'Commit Diff', Width = 800, Height = 600, Position = poScreenCenter, BorderStyle = bsSizeable.
  - `PanelHeader: TPanel` at top (Align = alTop, Height ~120): contains labels for hash, author, date, and a TMemo for full commit message (read-only, ~3 lines).
  - `PanelToolbar: TPanel` below header (Align = alTop, Height ~30): contains `ButtonToggleScope: TButton` with Caption 'Show Full Commit Diff' and a `LabelLoading: TLabel` with Caption 'Loading...' (initially visible during async fetch).
  - `RichEditDiff: TRichEdit` (Align = alClient): Font.Name = 'Consolas', Font.Size = 10, ReadOnly = True, WordWrap = False, ScrollBars = ssBoth, HideSelection = False.
- `class procedure ShowDiff(const ACommitHash, ARepoRoot, ARelativeFilePath: string; const ALineInfo: TBlameLineInfo)`:
  - Creates form, loads size from `BlameSettings.DiffDialogWidth/Height`.
  - Populates header labels from ALineInfo (hash, author+email, date).
  - Checks `CommitDetailCache.TryGet(ACommitHash)`:
    - If cached and `Fetched`: populate message memo and diff RichEdit immediately.
    - Otherwise: show loading label, call `FetchCommitDetailAsync`, on callback: populate and hide loading label.
  - `ShowModal`.
  - On close: save `BlameSettings.DiffDialogWidth := Width; BlameSettings.DiffDialogHeight := Height; BlameSettings.Save`.
  - `Free`.
- `procedure LoadDiffIntoRichEdit(const ADiff: string)`:
  - Iterate lines. Per line:
    - Starts with '+' (not '+++'): green color (light green `RGB(144, 238, 144)` for dark theme, `clGreen` for light).
    - Starts with '-' (not '---'): red color (light red `RGB(255, 150, 150)` for dark theme, `clRed` for light).
    - Starts with '@@': blue color (`clSkyBlue` for dark, `clBlue` for light).
    - Otherwise: default font color.
  - Use `RichEditDiff.SelStart/SelLength/SelAttributes.Color/SelText` pattern.
  - For large diffs (>5000 lines): truncate with "[Showing first 5000 lines]" notice.
  - Call `RichEditDiff.Lines.BeginUpdate/EndUpdate` around the whole operation.
- Theme adaptation: detect dark/light same as popup (DeriveAnnotationColor luminance check). Set form Color, RichEdit Color, label colors accordingly.
- Scope toggle button: default shows `FileDiff`. Clicking button switches to `FullDiff` (from cache) and changes button caption to "Show Current File Only". Toggle back and forth. If the alternate diff is not yet fetched (cache miss for FullDiff field), trigger async fetch.
