**Then implement** `src/DX.Blame.Git.Blame.pas`:

Export `procedure ParseBlameOutput(const AOutput: string; var ALines: TArray<TBlameLineInfo>)`.

Parser state machine (per research Pattern 3):
1. Split output by #10 (LF). Handle possible #13#10 by trimming #13 from line ends.
2. Scan for header line: length >= 40 and first char in ['0'..'9','a'..'f'].
3. Parse header: `Copy(Line, 1, 40)` for hash. Parse space-separated integers after hash for orig-line and final-line.
4. Read key-value pairs: lines starting with `author `, `author-mail `, `author-time `, `summary ` etc. Use `Copy(Line, PrefixLen+1, MaxInt)` to extract values.
5. `author-time` value: convert via `UnixToDateTime(StrToInt64Def(LValue, 0))` from System.DateUtils.
6. Stop key-value reading when a line starts with #9 (TAB = content line). Skip it.
7. Set `IsUncommitted := (LInfo.CommitHash = cUncommittedHash)`. If uncommitted, override Author to `cNotCommittedAuthor`.
8. Append to TList<TBlameLineInfo>, convert to array at end.

Uses: `DX.Blame.Git.Types`, `System.SysUtils`, `System.DateUtils`, `System.Generics.Collections`.

**Then implement** `src/DX.Blame.Cache.pas`:

```pascal
TBlameCache = class
private
  FLock: TCriticalSection;
  FData: TObjectDictionary<string, TBlameData>;
  function NormalizePath(const APath: string): string; // LowerCase
public
  constructor Create;
  destructor Destroy; override;
  procedure Store(const AFileName: string; AData: TBlameData);
  function TryGet(const AFileName: string; out AData: TBlameData): Boolean;
  procedure Invalidate(const AFileName: string);
  procedure Clear;
  function Contains(const AFileName: string): Boolean;
end;
```

Implementation details:
- `FData` is `TObjectDictionary<string, TBlameData>` with `doOwnsValues` -- the cache owns the TBlameData instances and frees them on removal/clear/destroy.
- `NormalizePath`: `Result := LowerCase(APath)` -- per research anti-pattern guidance, always lowercase for consistent lookup.
- All public methods acquire `FLock` in try/finally: `FLock.Enter; try ... finally FLock.Leave; end;`
- `Store`: if key exists, dictionary's `AddOrSetValue` will free the old value (doOwnsValues). Set new value.
- `Invalidate`: `FData.Remove(NormalizePath(AFileName))` -- TObjectDictionary frees the value.
- `Clear`: `FData.Clear` -- frees all values.

Uses: `DX.Blame.Git.Types`, `System.SyncObjs`, `System.Generics.Collections`, `System.SysUtils`.
