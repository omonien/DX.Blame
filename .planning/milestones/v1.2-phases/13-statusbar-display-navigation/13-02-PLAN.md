**1. Extend NavigateToRevision signature:**

Change the interface declaration from:
```pascal
procedure NavigateToRevision(const AFileName: string;
  const ACommitHash: string; const ARepoRoot: string);
```
To:
```pascal
procedure NavigateToRevision(const AFileName: string;
  const ACommitHash: string; const ARepoRoot: string;

```

The default `= 0` ensures backward compatibility — no existing callers need changes.

**2. Add auto-scroll after OpenFile in NavigateToRevision implementation:**

After the `LActionServices.OpenFile(LTempFile)` call (line 162), add:

```pascal
// Auto-scroll to source line if requested
if ALineNumber > 0 then
begin
  var LModuleServices: IOTAModuleServices;
  var LModule: IOTAModule;
  var LSourceEditor: IOTASourceEditor;
  var LEditView: IOTAEditView;
  var LEditPos: TOTAEditPos;

  if Supports(BorlandIDEServices, IOTAModuleServices, LModuleServices) then
  begin
    LModule := LModuleServices.FindModule(LTempFile);
    if LModule <> nil then
    begin
      for var i := 0 to LModule.GetModuleFileCount - 1 do
      begin
        if Supports(LModule.GetModuleFileEditor(i), IOTASourceEditor, LSourceEditor) then
        begin
          LSourceEditor.Show;
          LEditView := (BorlandIDEServices as IOTAEditorServices).TopView;
          if LEditView <> nil then
          begin
            LEditPos.Col := 1;
            LEditPos.Line := ALineNumber;
            LEditView.SetCursorPos(LEditPos);
            LEditView.Center(ALineNumber, 1);
            LEditView.Paint;
          end;
          Break;
        end;
      end;
    end;
  end;
end;
```

Note on Delphi 13 inline vars: The code above uses inline `var` declarations inside the `if ALineNumber > 0` block. If the project style avoids inline vars, declare them in the main var block of NavigateToRevision instead.

**Timer fallback:** Per research Open Question 1, `FindModule` may return nil if OpenFile is asynchronous. If this happens in practice, add a one-shot TTimer fallback. For now, implement the direct path — if FindModule returns nil, the file simply opens at line 1 (graceful degradation). The timer can be added in a gap closure plan if testing reveals the issue.

**3. Pass ALineNumber from OnRevisionClick:**
