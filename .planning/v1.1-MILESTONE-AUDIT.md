### Integration Issues

**MISS-1 (Medium): Retry timer not cleared on project switch**
- **Location:** `src/DX.Blame.Engine.pas:467` — `HandleBlameError`
- **Issue:** Retry `TTimer` created ad-hoc, never added to `FDebounceTimers`. `ClearAllTimers` and `OnProjectSwitch` cannot cancel it.
- **Impact:** After a blame error + project switch, `DoRetryBlame` fires with stale context. Mitigated by `IsSourceFile` path check.
- **Affected:** VCSA-05, HGB-01, HGB-04
- **Fix:** Track retry timers in a dedicated collection, cancel in `ClearAllTimers`.

**MISS-2 (Low): FVCSNotified not reset on project switch**
- **Location:** `src/DX.Blame.Engine.pas:388` — `OnProjectSwitch`
- **Issue:** `FVCSNotified` never reset to `False`, suppressing "No VCS repository detected" message on subsequent no-VCS projects.
- **Affected:** VCSD-05
- **Fix:** Add `FVCSNotified := False` before `Initialize` in `OnProjectSwitch`.
